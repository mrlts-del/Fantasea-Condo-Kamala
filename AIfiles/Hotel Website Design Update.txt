# Hotel Website Design Update

## Project Context
- **Framework**: Next.js 14 with TypeScript
- **Styling**: Tailwind CSS
- **File Structure**: App router structure
- **Logo**: FantaseaCondoLogo.png (contains teal, coral/pink, and cream colors)

## Brand Color Palette Guideline
```
- **Primary Colors (extracted from logo):**
  - **Teal/Turquoise**: #4A9B9B (main brand color - for headers, icons, borders)
  - **Coral/Pink**: #FF7B7B (accent color - for CTAs, highlights, buttons)
  - **Cream/Beige**: #F5F1EB (background sections, cards)

- **Supporting Colors:**
  - **Deep Teal**: #2D7A7A (hover states, darker elements)
  - **Light Coral**: #FFB3B3 (button hover states, light accents)
  - **Warm White**: #FFFFFF (main background, text areas)
  - **Charcoal**: #2C3E50 (primary text color)
  - **Light Gray**: #F8F9FA (section dividers, subtle backgrounds)

- **Color Application Guidelines:**
  - Use **Teal (#4A9B9B)** for: navigation, section headings, amenity icons, room card borders
  - Use **Coral (#FF7B7B)** for: CTA buttons, "Book Now" buttons, special offers, star ratings
  - Use **Cream (#F5F1EB)** for: alternating section backgrounds, testimonial cards
  - Use **Deep Teal (#2D7A7A)** for: button hover states, active navigation items
  - Maintain 60% neutral, 30% teal, 10% coral ratio throughout the design
```

## Technical Implementation Guidelines

### Code Quality Requirements
- Use TypeScript strictly (no any types)
- Implement responsive design using Tailwind breakpoints
- Add proper error handling for image loading
- Use Next.js Image component for optimized images
- Implement lazy loading for below-fold content

### Testing Checklist for Each Phase
- [ ] Desktop view (1920px, 1440px, 1024px)
- [ ] Tablet view (768px)
- [ ] Mobile view (375px, 320px)
- [ ] Touch interaction testing
- [ ] Image loading performance
- [ ] Color contrast accessibility (4.5:1 minimum)

### Performance Targets
- Page load time: < 3 seconds
- Image optimization: < 150KB per image
- Mobile-first responsive design
- Proper semantic HTML structure

## Validation Process
After completing each phase:
1. Test on multiple screen sizes
2. Verify brand colors are applied correctly
3. Check touch targets are minimum 48px
4. Validate image loading and responsiveness
5. Confirm text readability and contrast ratios
